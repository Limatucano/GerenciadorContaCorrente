# Teste para dev android

#### Um app para gerenciamento de contas correntes, nele é possivel:
- Ver saldo
- Extrato
- Saque
- Depósito
- Transferência
- Solicitar visita do gerente
- Trocar de usuário

#### foi usado a arquitetura MVVM, ROOM e viewBinding
